========================================================
  OYUN ÇEVİRİ MOTORU v2.0
  Kullanım Kılavuzu — Kurulumdan En Verimli Kullanıma
========================================================

─────────────────────────────────────────────────────────
  BÖLÜM 1: İLK KURULUM (Sıfır Bilgisayar / Format Sonrası)
─────────────────────────────────────────────────────────

Bu bölümü yalnızca ilk kez kuruyorsanız okuyun.

  TIKLAMA SIRASI:

  1. Bu klasörü bilgisayarınıza kopyalayın.
     (USB, bulut sürücü veya ağ paylaşımından)

  2. "tam_kurulum.bat" dosyasına sağ tıklayın.
     → "Yönetici olarak çalıştır" seçin.
     → Evet / İzin Ver'e tıklayın.

  3. Siyah CMD penceresi açılır.
     Otomatik olarak şunları yapar:
       a. Python 3.11 yüklü değilse indirir ve kurar (~25 MB)
       b. Gerekli paketleri internetten indirir (~150 MB)
       c. Windows OCR modüllerini kurar
       d. Kurulum bitince program otomatik açılır

  4. İNTERNET BAĞLANTISI GEREKLİDİR.
     Kurulum internet olmadan yapılamaz.

  TAHMINI SÜRE: 3-10 dakika (internet hızınıza göre değişir)

  NOT: Bu adımları yalnızca bir kere yapmanız yeterlidir.
       Bir dahaki açılışta sadece "calistir.vbs" kullanın.


─────────────────────────────────────────────────────────
  BÖLÜM 2: HER ZAMANKI KULLANIM
─────────────────────────────────────────────────────────

  TIKLAMA SIRASI:

  1. "calistir.vbs" dosyasına çift tıklayın.
     CMD penceresi açılmaz. Program doğrudan başlar.

  2. Program açılırken sağ üstte "Donanım Taranıyor..."
     yazısı görünür. Bu birkaç saniye sürer.
     → Yazı "Windows OCR" veya "EasyOCR GPU" olarak
       değişince sistem hazır demektir.

  3. Oyununuzu başlatın ve arka planda bırakın.


─────────────────────────────────────────────────────────
  BÖLÜM 3: ALTYAZI BÖLGE SEÇİMİ
─────────────────────────────────────────────────────────

  En önemli adım — doğru bölge seçilmezse çeviri çalışmaz.

  TIKLAMA SIRASI:

  1. Oyununuzu başlatın.
  2. Altyazının göründüğü bir sahneye gelin ve DONDURUN
     (örn: bir diyalogda, pause menüsünde)
  3. Klavyeden Ctrl+1 tuşuna basın.
     → Ekran kararır ve imleç değişir.
  4. Farenizi altyazının etrafına sürükleyerek seçim yapın.
     → Altyazının biraz dışından başlayıp biraz içeride bitirin.
     → Çok geniş seçim yapmaktan kaçının (OCR gürültü okur)
  5. Ctrl+2 ile çeviriyi başlatın.

  İPUCU: Altyazı alanı değiştiyse Ctrl+1 ile yeniden seçin.


─────────────────────────────────────────────────────────
  BÖLÜM 4: KISA YOL TUŞLARI
─────────────────────────────────────────────────────────

  Ctrl+1  → Altyazı bölgesi seç (ekran tarama alanı)
  Ctrl+2  → Çeviriyi Başlat / Durdur
  Ctrl+3  → Altyazı overlay'ini ekran ortasına sıfırla
  Ctrl+4  → İkinci bölge (HUD/bilgi metni) seç
  Ctrl+5  → İkinci bölgeyi Başlat / Durdur
  Ctrl+Q  → Programı kapat


─────────────────────────────────────────────────────────
  BÖLÜM 5: KENAR ÇUBUĞU AYARLARI (Sol Panel)
─────────────────────────────────────────────────────────

  Tüm başlıklara tıklanarak açılıp kapatılabilir.

  ┌─ DİL YÖ NÜ ─────────────────────────────────────────┐
  │  Kaynak Dil  : Oyundaki yazıların dili               │
  │               Çoğu oyun için "İngilizce (EN)"        │
  │  Hedef Dil   : Çevrilecek dil — varsayılan Türkçe    │
  └──────────────────────────────────────────────────────┘

  ┌─ ALTYAZI GÖRÜNÜMÜ ────────────────────────────────────┐
  │  Yazı Tipi   : Segoe UI önerilir (Windows uyumlu)     │
  │  Renk        : Sarı — en okunabilir                   │
  │               Beyaz — parlak sahnelerde kaybolabilir  │
  │  Altyazı     : S(14) / M(18) / L(24) / XL(32)        │
  │  Boyutu        Büyük ekran için XL seçin              │
  │  + / −       : Piksel piksel ince ayar                │
  │  B Butonu    : Kalın yazı aç/kapat                    │
  └──────────────────────────────────────────────────────┘

  ┌─ ARAYÜZ ÖLÇEĞİ ──────────────────────────────────────┐
  │  4K / 2K ekran kullanıyorsanız "Büyük" veya "Dev"    │
  │  seçin — tüm arayüz orantılı büyür.                  │
  │  Full HD için "Normal" yeterlidir.                    │
  └──────────────────────────────────────────────────────┘

  ┌─ TARAMA HIZI ─────────────────────────────────────────┐
  │  Yavaş  : 1.0 sn aralık — düşük CPU kullanımı        │
  │  Normal : 0.5 sn — standart kullanım (önerilen)       │
  │  Hızlı  : 0.25 sn — hızlı diyaloğa yetişir           │
  │  Süper  : 0.1 sn — maksimum, yüksek CPU kullanımı    │
  └──────────────────────────────────────────────────────┘

  ┌─ KISAYOLLAR ──────────────────────────────────────────┐
  │  Referans için listelenmiştir, ayarlanamaz.           │
  └──────────────────────────────────────────────────────┘


─────────────────────────────────────────────────────────
  BÖLÜM 6: OCR MOTORU SEÇİMİ (Ana Ekran Ortası)
─────────────────────────────────────────────────────────

  Üç motor arasından biri seçilir. Gri görünen motorlar
  henüz kurulmamış demektir — tıklanamaz.

  ┌─ Windows OCR (Her zaman hazır) ──────────────────────┐
  │  • Kurulum gerektirmez, anında çalışır               │
  │  • CPU %1-2, RAM ~50 MB kullanır                     │
  │  • İnternet gerektirmez                              │
  │  • Standart fontlarda güvenilir                      │
  │  • Süslü/grafik fontlarda zorlanabilir               │
  └──────────────────────────────────────────────────────┘

  ┌─ EasyOCR GPU (Kurulum gerektirir) ───────────────────┐
  │  • NVIDIA ekran kartı gereklidir                     │
  │  • Kurulum: Kur butonuna tıklayın (~2.5 GB indirir)  │
  │  • GPU %40-80, VRAM ~1.5 GB kullanır                 │
  │  • En yüksek OCR doğruluğu — grafik fontlar dahil   │
  │  • İlk yüklemede ~20 saniye bekleme olur             │
  └──────────────────────────────────────────────────────┘

  ┌─ Yerel AI (Kurulum gerektirir) ──────────────────────┐
  │  • İnternet bağımsız ÇEVIRI motoru (OCR değil)      │
  │  • Kurulum: Kur butonuna tıklayın (~1.2 GB indirir)  │
  │  • CPU %5-15 veya GPU %10-30 kullanır               │
  │  • Kurulduktan sonra tamamen çevrimdışı çalışır      │
  └──────────────────────────────────────────────────────┘


─────────────────────────────────────────────────────────
  BÖLÜM 7: EN VERİMLİ KULLANIM İÇİN TAVSİYELER
─────────────────────────────────────────────────────────

  SENARYO A — Siyah şeritli altyazı (ör: Cyberpunk 2077)
    1. Oyun ayarları → Altyazı Arka Planı: Açık / Opak
    2. Altyazı boyutunu orta-büyük yapın
    3. Bölge seçerken şeritli alana sığın, dışarı taşmayın
    4. Motor: Windows OCR yeterlidir
    5. Tarama hızı: Normal

  SENARYO B — Arka plansız, sahne üstü yazı
    1. Altyazı rengi: Sarı seçin (beyaz kaybolabilir)
    2. Motor: EasyOCR GPU varsa onu seçin
    3. Tarama hızı: Hızlı (kaçan satırları yakalar)
    4. Bölge seçimini harf yüksekliğinden biraz geniş yapın

  GENEL
    • "Çeviri Geçmişi" paneli — tekrar okumanız gereken
      satırlar için kaydırarak bakabilirsiniz
    • Overlay metnini sürükleyerek istediğiniz konuma
      taşıyabilirsiniz
    • Ctrl+3 ile overlay'i merkeze sıfırlayabilirsiniz
    • Ayarlarınızı "Ayarları Kaydet" ile kaydedin —
      program her açılışta bu ayarları yükler


─────────────────────────────────────────────────────────
  BÖLÜM 8: SIKÇA SORULAN SORULAR
─────────────────────────────────────────────────────────

  S: Program açılıyor ama çeviri gelmiyor.
  C: Ctrl+1 ile bölge seçtiğinizden emin olun.
     Ardından Ctrl+2 ile başlatın.

  S: Altyazı gözüküyor ama hep aynı satır kalıyor.
  C: TextStabilizer çalışıyor olabilir (konsensüs bekliyor).
     3-4 saniye bekleyin, yeni satır gelince güncellenir.

  S: Çeviri yanlış veya saçma çıkıyor.
  C: OCR yanlış okuyor demektir. Bölge seçimini daraltın
     ve EasyOCR GPU motorunu deneyin.

  S: Program çok yavaş / donuyor.
  C: Tarama hızını "Yavaş" veya "Normal" yapın.

  S: Windows "bilinmeyen yayıncı" uyarısı veriyor.
  C: Normal — "Yine de çalıştır"a tıklayın.
     Bu program dijital imzalı değildir, bu normaldir.

  S: Keyboard hatası çıkıyor, kısayollar çalışmıyor.
  C: calistir.vbs yerine tam_kurulum.bat dosyasını
     "Yönetici olarak çalıştır" ile başlatın.
